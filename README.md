# Developer Manual

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/benwesson/INST_377_Group_Final_Repo.git
    ```
2. Navigate to the project directory:
    ```bash
    cd INST_377_Group_Final_Repo
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Running the Application
1. Start the server:
    ```bash
    npm init
    Is this Ok? (yes) yes
    npm start
    ```
